# Student page export

Static, self-contained exports of the marketing `/students` page (coral/cream
design), generated from a production render with all CSS inlined/bundled and
fonts loaded from Google Fonts. No server or JavaScript required — open in any
browser.

| File | What it is |
|------|------------|
| `resumetrakr-students-page.html` | Single self-contained file — **CSS embedded** in a `<style>` block. Just open it. |
| `resumetrakr-students-page.zip` | Zip of the two files below (HTML + external CSS). |
| `students-index.html` | The page markup; links `students-styles.css`. |
| `students-styles.css` | The combined page CSS (with a Google Fonts `@import`). |

To download from GitHub, open a file and use **Download raw file** (or download
the `.zip` directly).

> Note: these are point-in-time snapshots for sharing/preview — they are not
> wired into the app build and won't update automatically when the live page
> changes.
